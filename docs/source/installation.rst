.. Installation doc

Prerequisites
=============

* Python installed

Installation
============

I recommend to use `virtualenv`_ with Python 3.

Pypi package
------------

Install via pip:

.. code-block:: bash

    pip install rasahub


Bleeding-Edge Version
---------------------

Install from sources:

.. code-block:: bash

    git clone https://github.com/DServSys/rasahub.git
    cd rasahub
    pip install -e .


.. _virtualenv: https://virtualenv.pypa.io/en/stable/
